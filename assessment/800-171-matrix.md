# NIST SP 800-171 → Lab Implementation Matrix

Starter matrix mapping each control **family** to the lab component that
implements it and the evidence artifact that proves it. Expand each row to the
individual controls (110 total) as you build. Status: `Met` / `Partial` /
`Planned`.

| Family | Lab implementation | Evidence artifact | Status |

| **AC** – Access Control | Tiered admin, PAW, DAC on CUI share, session lock GPO | GPO export, share ACLs, `evidence/AC/` | Planned |

| **AU** – Audit & Accountability | Wazuh centralized logging, time sync, control-tagged rules | Wazuh rules, log samples, `evidence/AU/` | Planned |

## SPRS scoring note

Level 2 self-assessment scores out of 110 using the DoD Assessment Methodology
(controls are weighted 1, 3, or 5 points; you start at 110 and subtract).
Record the running score and each POA&M item in `evidence/sprs/`.
